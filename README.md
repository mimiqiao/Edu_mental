# 儿童心理锦囊铺

> 版本：0.1.0

一个基于AI的儿童心理健康咨询平台，为家长和教育工作者提供专业的儿童心理健康指导。

## 功能特点

- 智能问答：通过AI技术提供个性化的儿童心理健康咨询
- 心理知识库：提供丰富的儿童心理学知识内容
- 简洁的用户界面：直观易用的交互设计

## 技术架构

- 前端：React
- 后端：Node.js
- AI：OpenRoute API

## 快速开始

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 访问应用
打开浏览器访问 http://localhost:3000
